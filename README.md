# Natular-Language-Processing
this laboratory is the solve homework for the course on Natural Language Processing in unipd. It was asked to solve $2$ problems:
- the first was to pre process a corpus and count the occurrences of each word. The professor then gave us the code to fit a Zipf's law curve
- the second was about solve a ML problem in which we had to vectorize a corpus and apply a model to do sentimental analysis. The model used was logistic regression; the vectorizer was CountVectorizer and i personally defined my customed tonenizer. The model can be improved by picking another vectorizer and using lemmatization.
